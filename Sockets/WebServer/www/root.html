You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=1&num2=2 -- returns result of multiplication
  • /github?query=users/amehlhase316/repos -- returns information about public repos for user
  • /age?name1=josh&name2=john -- returns prediction of the age of a person given their name(s)
  • /numbers?num=1&cat=trivia -- returns interesting number fact for entered number in categories (options are “trivia”, “math”, or “year”)
